# Memory Card — викторина на PyQt5

Интерактивная викторина с выбором ответа, созданная с использованием библиотеки PyQt5. Программа задаёт вопросы с четырьмя вариантами ответа, проверяет правильность выбора и ведёт подсчёт результатов.

## Функциональность

Программа реализует следующие возможности:
* отображение случайного вопроса из списка;
* перемешивание вариантов ответов для разнообразия;
* проверка правильности выбранного ответа;
* отображение результата (правильно/неправильно) и правильного ответа;
* подсчёт количества правильных ответов и общего количества вопросов;
* расчёт процента правильных ответов;
* переход к следующему случайному вопросу.

## Технологии

* Python 3.x
* PyQt5 (графический интерфейс)
* Стандартные библиотеки Python: `random`
